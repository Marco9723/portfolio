LSTM Gear

LSTM GEAR is a new plugin that emulates electric guitar effects using neural models in C++ through blackbox modelling of analog pedals.
This work also contains a report on the new and promising RTNeural library for neural models implemented in C++.
GitHub | YouTube

"Kinetic Sonic Performer" is a project built on a full-fledged model designed to foster continuous interaction between the user and the system.
